# GrowPlus+ — Digital Marketing Agency Website

A fully functional, multi-page dark-themed website built with pure HTML, CSS, and JavaScript. No frameworks, no build tools — just open and it works.

## 📁 File Structure

```
growplus/
├── index.html          ← Home page
├── about.html          ← Who We Are
├── why.html            ← Why We're Different
├── services.html       ← Services Overview
├── seo.html            ← SEO Service page
├── performance.html    ← Performance Marketing page
├── social.html         ← Social Media Marketing page
├── affiliate.html      ← Affiliate Marketing page
├── brand.html          ← Brand Marketing page
├── influencer.html     ← Influencer Marketing page
├── content.html        ← Content Marketing page
├── blog.html           ← Blog with filter
├── updates.html        ← News & Updates
├── contact.html        ← Grow With Us / Contact
├── landing.html        ← All Services Landing Page (2 forms)
├── css/
│   └── style.css       ← All styles
└── js/
    ├── layout.js       ← Shared nav + footer (injected on every page)
    └── main.js         ← All JavaScript (cursor, canvas, animations, etc.)
```

## 🚀 How to Deploy on GitHub Pages

1. Create a new repository on GitHub (e.g. `growplus-website`)
2. Upload all files maintaining the folder structure above
3. Go to **Settings → Pages**
4. Set Source to **Deploy from a branch → main → / (root)**
5. Your site will be live at `https://yourusername.github.io/growplus-website/`
